# Student Assignment Tracker

A full-stack web application that helps students organize assignments, track deadlines, and manage coursework.

## Features
- User authentication and personalized accounts
- Create, edit, and delete assignments
- Categorize assignments by type (Homework, Classwork, Quiz, Exam)
- Add notes to assignments
- Mark assignments as completed
- Calendar view for assignment deadlines

## Technologies
- Python
- Flask
- SQLAlchemy
- SQLite
- HTML/CSS
- Bootstrap
